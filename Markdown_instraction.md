## Списки
XN,s добавить ненумерованный список, необходимо пункты выделить * Вот так: 
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Пункт 1
2. Пункт 2
